Introducción¶

Contexto¶

El terrorismo es algo que lamentablemente ha estado en nuestra sociedad desde hace dcadas. Se define terrorismo como el uso sistemtico del terror usado por distintas organizaciones o individuos con el fin de promocionar sus objetivos, ya sean polticos, econmicos, religiosos o personales entre otros. Da a da se escuchan noticias referidas al tema en distintos lados del mundo, por lo que es interesante tener a disposicin el registro histrico de estos eventos. Su estudio ayudara a descubrir patrones de comportamientos o encontrar nueva informacin sobre eventos pasados para aprender de ellos, con la motivacin de poder mejorar polticas de prevencin de estos sucesos, que muchas veces significan prdidas de vidas, destruccin de infraestructura, flora y/o fauna.

Motivación¶

La principal motivacin que lleva a escoger y estudiar este Data Set es encontrar y comprender patrones existentes en los ataques terroristas, visualizar la evolucin de estos a lo largo de los aos y encontrar tendencias y por ultimo, estudiar las implicancias existentes en los ataques terroristas, ya sea economica, poltica y socialmente. Esto dado que al tener mas informacin de esto, se podran establecer protocolos de prevencin.

import datetime
import numpy as np

# Merge iyear, imonth, iday column into a timestamp column within a timestamp format
df['timestamp'] = df[['iyear', 'imonth', 'iday']].apply(lambda s: datetime.datetime(*s), axis=1)
df = df.loc[:, ~df.columns.isin(['iyear', 'imonth', 'iday'])]

# Merge extended, resolution columns into end_date and duration (days) column
end_date  = pd.to_datetime(df['resolution'], format='%Y-%m-%d')
end_date = np.where(pd.isnull(df['resolution']), df['timestamp'], end_date)
df['end_date'] = end_date
df['duration'] = (df['end_date'] - df['timestamp']).apply(lambda s: s.days + 1)
df = df.loc[:, ~df.columns.isin(['extended', 'resolution'])]

# Filter out incidents where duration has a non positive value
df = df[df.duration > 0]

# Add an ISO-3166-1 column with 3-letter country code for each entry, used to plot maps in certain libraries.
def search_country_code(country_name):
    try:
        return pycountry.countries.search_fuzzy(country_name)[0].alpha_3
    except:
        return 'UNK'

nkill tiene relacin con nhostkid con un valor de -0.399, esto significa que a medida que aumenta el nmero de muertes, disminuye el nmero de rehenes, lo cul en primera instancia no se piensa pero brinda un pilar inicial para poder estimar si es que un grupo tiende a estar en un atentado que hay altas muertes, entonces la tasa de rehenes o secuestrados muertos aumenta tambien.\ nhostkid posee una correlacin de -0.946 con nrealized, lo que significa que si el nmero de secuestros o rehenes aumenta, entonces disminuye con creces el nmero de personas que sobrevive al incidente y que fueron tomadas como rehen.

nkill posee una correlacin de 0.7 con nwound, lo que dice que aproximadamente un 70% de las personas que fueron heridas en el incidente, finalmente murieron.

Anlisis como estos servirn en un futuro para generar predicciones y asociaciones no triviales en este dataset, pues aparte de describir que significa cada valor correlacional (lo que sera muy largo y poco productivo), permite usar dichos valores para por ejemplo, asociar un peso a un conjunto de atributos en una red neuronal destinada a hacer clasificaciones en este dataset, siendo un valor en principio oculto pero que sirva como ladrillo para generar una prediccin bien fundamentada.

Preguntas y problemas¶

Una vez viendo el panorama del dataset limpio y analizado, como grupo se generaron las siguientes preguntas e hiptesis que se estiman pueden ser respondidas y validadas con herramientas de minera de datos:

  • Se puede establecer un mapa de calor de donde se concentran los atentados terroristas? Y cmo ha variado este en el tiempo?
  • Existe alguna relacin referente a la tasa de xito de estos atentados frente a alguna de las caractersticas de la base de datos?
  • Cmo han variado en el tiempo los ataques terroristas en cuanto armas/tipo de ataque/objetivos?
  • Es posible establecer alguna relacin entre zonas geogrficas y si el ataque es domstico o internacional?
  • Es posible que un cierto grupo de atributos pertenecientes a una instancia tengan una alta correlacin con el acierto de un ataque?
  • Existe alguna relacin entre la zona del ataque y la fecha del atentado?
  • Hay algn tipo de relacin entre la zona geogrfica del ataque con el motivo de ste? (ejemplo, ataques con ndole religioso con ms frecuentes en ciertos sectores)
  • Es posible, con los datos actuales, encontrar zonas de sospecha de aumento de actos terroristas?
  • Verificar las reas de dominio de distintos grupos a escala global.
  • Existe relacin entre el pais y el tipo de acto terrorista que ha sufrido de manera mas frecuente?
  • Patrn o tendencia de alguna nacionalidad en cometer ataques terristas sobre otro.
  • Comportamiento del terrorismo a lo largo de los aos.

Para el prximo hito se espera poder combinar este data set con algunas variables geopoliticas de los paises, como el sistema politico, el poder economico, las religiones e ideologas predominantes, hitos histricos que ocurrieron cercano al ataque terrorista, etc.
